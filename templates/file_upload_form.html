To upload files using cURL you can run: curl -X POST https://{{ BASEURL }}/{{ DUSER }}/{{ DPROJECT }}/r/notebookSession/{{ DRUNID }}/success -F file=@PATH TO FILE




Uploaded files will be stored under the systempath {{ MYPATH }}.

Or you can find the files in the Domino UI under the local dataset of project {{ DPROJECT }}

When a large upload size is needed make sure to modify Domino central config key "com.cerebro.domino.computegrid.kubernetes.apps.nginx.clientBodyMaxSizeMiB" , the default value is 25 MiB .
